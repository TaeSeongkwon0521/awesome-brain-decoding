# Awesome Brain Decoding

A curated list of papers, datasets, benchmarks, and code resources for brain decoding research.

## Topics

- fMRI-based Brain Decoding
- Natural Scenes Dataset (NSD)
- fMRI-to-Image Reconstruction
- Diffusion-based Reconstruction
- Vision-Language Alignment
- Brain-to-Text / Brain-to-Caption
- Cross-Subject Brain Decoding
- NeuroAI

## Collections

- [Awesome Brain Decoding with NSD](./awesome-brain-decoding-nsd/README.md)

## Purpose

This repository aims to organize recent research trends in brain decoding, especially focusing on visual decoding from fMRI signals.
